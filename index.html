The Dreamer

November 2024

Link: https://dacevedo11.itch.io/the-dreamer

GitHub Repository: https://github.com/dacevedo11/the-dreamer

Description:
This project was made for CTIN 534 with Peter Brinson at USC. The goal of The Dreamer assignment is to make a “Bad Dream Escape Room”. This game aims to blend elements of reality and fiction, placing the player in the role of a veteran battling PTSD. They encounter bizarre phenomena, like exploding Teddy Bears, highlighting the fragmented boundaries between their inner turmoil and the outside world. This game is currently a work in progress, and is being made with Unity.

Turtle Tank Defense

October 2024

Link: https://dacevedo11.itch.io/turtle-tank-defense

GitHub Repository: url

Description:
This game was my midterm project for CTIN 583 with Debbie Yuen at USC. The assignment was to make a game that tackles a social justice issue. I chose to make a game that educates players about the effects of littering and plastic pollution. This is a tower defense game where you place Turtle Tanks to destroy plastic waste before it can reach the ocean. Waves of trash spawn each round, and the amount of trash increases with each round. Made in Unity.

Mystery Detective

October 2024

Link: https://dacevedo11.itch.io/meaningful-play-assignment

Description:
This project was made for the Meaningful Play assignment in CTIN 541 with Tracy Fullerton. My group and I aimed to create an immersive experience full of tension and drama. We developed a “whodunit” murder mystery game where players assume the roles of five characters, each with ties to the murder. As they unravel clues, players must either work together to uncover the true killer or devise strategies to avoid suspicion and escape detection.
