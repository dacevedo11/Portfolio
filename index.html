Games by Daniel Acevedo

The Dreamer

November 2024

Link: https://dacevedo11.itch.io/the-dreamer

GitHub Repository: https://github.com/dacevedo11/the-dreamer

Description:
This game blends elements of reality and fiction, placing the player in the role of a veteran battling PTSD. With the assigned theme of "Bad Dream Escape Room", they encounter bizarre phenomena — like teddy bears that suddenly explode — highlighting the fragmented boundaries between their inner turmoil and the outside world. This game is in early development and is being made in Unity 3D.

Turtle Tank Defense

October 2024

Link: https://dacevedo11.itch.io/turtle-tank-defense

Description:
This is a tower defense game where you place Turtle Tanks to destroy plastic waste before it can reach the ocean. Waves of trash spawn each round, and the amount of trash increases with each round. My motivation for making this game was to educate players about the effects of littering and plastic pollution. Made in Unity 3D.

Kitchen Chaos

September 2024

Link: https://dacevedo11.itch.io/kitchen-chaos

Description:
This was Project 1 for CTIN 583 at USC. We followed a YouTube tutorial to create a game, similar to Overcooked, in Unity 3D. The goal for this project was to learn the basics of Unity and C# programming. The game itself is incomplete, but I learned how to use prefab variants, generics, and template classes in Unity.

Up The River Mod

September 2024

Link: https://dacevedo11.itch.io/up-the-river-mod

Description:
This was a group assignment in Tracy Fullerton’s section of CTIN 541. Our goal was to create a modified version of Up The River. My group and I created a racing themed variant aimed at giving players the option to either boost themselves forward or sabotage other players to slow them down. I contributed to the design of the system, mechanics, and game board. .

The Investigator

September 2024

Link: https://drive.google.com/file/d/1BaPx38c_3exfeXmAh3eQxNY7BO4Pl9zJ/view?usp=drive_link

Description:
This was a group assignment for CTIN 534 with Peter Brinson. The objective was to create a murder-mystery type game that included a “ghost” appearance. We made this game using Unity 2D, and it centers around a museum security guard who encounters a ghost. The core mechanic in this game is exploration through use of a flashlight.
